# LeetCode刷题（简单难度）
## 1.两数之和

    给定一个整数数组和一个目标值，找出数组中和为目标值的两个数。
    你可以假设每个输入只对应一种答案，且同样的元素不能被重复利用。

示例：

    给定 nums = [2, 7, 11, 15], target = 9
    因为 nums[0] + nums[1] = 2 + 7 = 9
    所以返回 [0, 1]

代码（JavaScript）
```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function(nums, target) {
    let i = 0, j = i + 1
    while(nums[i] + nums[j] !== target || j >= nums.length || i >= nums.length-1) {
        j++
        if (j === nums.length) {
            i++
            j = i + 1
        }
    }
    return [i, j]
};
```
